# Sorting tool with Java [Hyperskill Project](https://hyperskill.org/projects/45)
This project provides a tool for processing and sorting both textual and numerical data. The program analyzes input datasets, identifies the largest or most frequent elements, and performs various calculations to help make sense of large volumes of information.

Graduation project of Java Developer [course](https://hyperskill.org/courses/17-java-developer) confirmed by [certificate](https://hyperskill.org/certificates/87cd6646-22f7-4689-a018-334751a77cc8.pdf)
